# NucleBrain-CCTweaked-Reactor-Manager
A simple Bigger Reactors management program. Allows for control rod groupings.

# How To Use
  Build a reactor with n number of control rods.\n
  Label your control rods as GROUP#_NAME (Ex. 1_A, 12_ALPHA)\n
  Connect the reactor computer port to the back of the computer.\n
  run 'NucleBrain' from the computer.\n
